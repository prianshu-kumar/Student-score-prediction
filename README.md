# 📚 Student Score Prediction using Machine Learning

[![Streamlit App](https://img.shields.io/badge/Streamlit-Live_App-red?logo=streamlit)](https://student-score-prediction-prianshu.streamlit.app/)

A Machine Learning project that predicts student scores based on study-related factors using regression techniques.

---

## 🚀 Demo

Predict student scores interactively using a Streamlit web app.

---

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## 📊 Features

- Linear Regression model
- Model evaluation
- Interactive Streamlit UI
- Model persistence using Pickle

---

## ▶️ Run Locally

Clone repository:

```bash
git clone https://github.com/prianshu-kumar/Student-score-prediction.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Streamlit app:

```bash
streamlit run app/app.py
```
